Definitions and ontology scope

Ontology
A formal, machine-readable specification of concepts, relationships, and constraints within a domain.
Class
A category of things that share a common meaning, role, or set of characteristics.
Subclass
A more specific class whose instances are also instances of its parent class.
Object property
A relationship connecting an instance of one class to an instance of another class.
Datatype property
A property connecting an ontology instance to a literal value such as text, a date, or a duration.
Annotation property
Documentation metadata used to describe an ontology entity without defining its domain logic.
Domain
The class or classes whose instances may occur as the subject of a property.
Range
The class, classes, or datatype whose instances or values may occur as the object of a property.
Inverse property
A property that expresses the reverse direction of another object property.
OWL restriction
A class-level condition specifying how instances of a class must use a property, such as requiring at least one related component.
RDF triple
A statement represented as subject–predicate–object.
Transitive property
A relationship where A–B and B–C imply A–C, when the intended semantics support that inference.
